== Description ==

With this Plugin you can manage Notes in the Admin Menu. To write a new note click on the little note Icon in your Admin Menu.
You find the Icon right on the Top. Manage as much Notes as you want.
Of course you can delete an edit them.

If you want to share your Notes with other WordPress User, use the Sharing Function.
Share to all, to one user or to none.

This Plugin work with jQuery UI and Ajax.

== Installation ==

1. Upload the "admin-ajax-note" Folder with all files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use it.
